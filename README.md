Overview
This project demonstrates a complete end-to-end data analytics workflow, starting from raw data and ending with actionable business insights. It covers data loading, cleaning, exploratory data analysis (EDA), SQL-based analysis, dashboarding, and reporting.

Dataset
Source: Public / Provided dataset (CSV format)
Type: Transactional / Customer / Sales data
Size: Medium-sized (suitable for analytics & BI)
Key Features:
  Customer attributes
  Purchase details
  Discounts, categories, and monetary values
  Time-based fields


Tools & Technologies
Python: Data loading, cleaning, and EDA
Libraries: pandas
SQL: Business queries & aggregations
Databases: MySQL
Power BI: Interactive dashboard creation
Gamma: Business presentation (PPT)
Jupyter Notebook: Analysis workflow

Project Workflow
1. Data Loading
Imported raw CSV data into Python using pandas
Verified schema, data types, and basic structure

2. Exploratory Data Analysis (EDA)
Summary statistics and distributions
Outlier detection
Trend and category-level analysis
Business-focused questions explored

3. Data Cleaning
Handled missing values
Corrected data types
Removed duplicates
Standardized categorical values

4. SQL Analysis
Loaded cleaned data into SQL database
Wrote optimized SQL queries to answer business questions such as:
Revenue by customer segment
Discount impact on sales
Top-performing products and categories
Used subqueries, aggregations, and window functions

5. Dashboard Development
Built an interactive Power BI dashboard
Key KPIs, filters, and drill-downs included
Designed for business users, not analysts

6. Reporting & Presentation
Created a structured analytical report
Designed a PPT using Gamma to summarize:
  Problem statement
  Key insights
  Visual findings
  Business recommendations

Dashboard
Power BI Dashboard Highlights:
Revenue & sales KPIs
Category and product performance
Customer segmentation insights
Discount vs revenue impact
Interactive slicers for deep dives

Key Results & Insights
Identified high-revenue customer segments
Analyzed the real impact of discounts on revenue
Highlighted underperforming categories
Provided data-backed recommendations for business optimization
